To run this program:
0. Requirements:
	- An up to date version of the JDK/JRE installed
1. From the terminal, compile using the java compile command:
	javac Phase1.java
2. Once compiled, run Phase1.class by the java command:
	java Phase1 [FileNameWithExtension] [HiddenMessageType] #HiddenMessageType is 'text' or 'png'
	or
	java Phase1 [Path/To/FilenameWithExtension] [HiddenMessageType]  #if file is not in the same folder as Phase1.class
